# whimsy

Small things made for the joy of making them.

## elsewhere

A multi-city dashboard for places you are, have been, or might go.

Type a city name, pick from the dropdown, and it adds to the list with current
local time and weather. Click "See scenery" on any card to open a Pixabay
search for that location.

Open `elsewhere.html` in a browser. Cities persist via localStorage.

Uses [Open-Meteo](https://open-meteo.com/) for geocoding and weather (no API
key required) and Pixabay search URLs for scenery links.
